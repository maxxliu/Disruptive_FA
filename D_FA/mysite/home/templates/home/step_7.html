Step 7: Calculating Enterprise Value

 

Making Use of the Terminal Value

After finding the present value of future free cash flows (PV FCF), the second step is to find the present value of the terminal value (PV TV) that we have just calculated

 

PV Terminal Value = Terminal Value / (1+WACC)n

 

Enterprise Value = PV FCF + PV TV

    Where:
      PV FCF: Present value of free cash flows
      PV TV: Present value of the terminal value
