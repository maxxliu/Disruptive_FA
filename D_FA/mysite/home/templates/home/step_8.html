Step 8: Find the Share Price

 

Find Enterprise Value (EV):

EV = PV FCF + PV TV

 

Then, derive Equity Value from EV:

EV - Debt - Preferred Stock - Minority Interest + Cash = Equity Value

 

Last, but not least, divide Equity Value by shares outstanding:

Equity Value / Shares Outstanding = Price per Share

 

Typically, valuations are reported as ranges using a sensitivity table and visually in a football field diagram
